Setup Gmail App Password

  1. Go to your Google Account Management page.
  2. Select Security from the left-hand navigation panel.
  3. Under the "How you sign in to Google" section, ensure 2-Step Verification is enabled.
  4. Click on 2-Step Verification, scroll to the very bottom, and select App passwords.
  5. Enter a name for the app (e.g., ClientRadar AI) and click Create.
  6. Google will generate a 16-character code (e.g., abcd efgh ijkl mnop). Copy this code.
  7. Paste the 16-character code directly into the **Gmail App Password** field in the Settings connection form.
Why is this needed? Google restricts basic password access from third-party apps. An App Password grants a secure, isolated token for sending emails from your address without sharing your actual Gmail password.
